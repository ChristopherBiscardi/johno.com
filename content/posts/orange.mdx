## Of all the named colors, orange is the tastiest fruit

</Flex>

<Flex>

<div style={{ height: 200, width: 200, borderRadius: 9999, minWidth: 200, flexGrow: 1, backgroundColor: 'black', color: 'orange', fontWeight: 'bold', display: 'flex', alignItems: 'center', justifyContent: 'space-around', lineHeight: 1, fontSize: 80, marginRight: 20 }}>AAA</div>

When combined with `black`, `orange` is accessible. It results in a
**AAA** score and the contrast ratio is **10.6**.

</Flex>

<Flex>

<div style={{ height: 200, width: 200, borderRadius: 9999, minWidth: 200, flexGrow: 1, color: 'orange', border: '2px solid orange', fontWeight: 'bold', display: 'flex', alignItems: 'center', justifyContent: 'space-around', lineHeight: 1, fontSize: 80, marginRight: 20 }}>F</div>

When combined with `white`, `orange` isn't accessible. It results in an
**F** score and the contrast ratio is **1.9**. At first this might seem
like a bummer, but we can still use it for accents like borders.

</Flex>

<Flex>

<div style={{ height: 200, width: 200, minWidth: 200, borderRadius: 9999, flexGrow: 1, border: '16px solid orange', backgroundColor: 'black', color: 'orange', fontWeight: 'bold', display: 'flex', alignItems: 'center', justifyContent: 'space-around', lineHeight: 1, fontSize: 80, marginRight: 20 }}><code>#</code></div>

</Flex>

<Flex>

<div style={{ height: 200, width: 200, minWidth: 200, borderRadius: 9999, flexGrow: 1, backgroundColor: 'orange', color: 'black', fontWeight: 'bold', display: 'flex', alignItems: 'center', justifyContent: 'space-around', lineHeight: 1, fontSize: 80, marginRight: 20 }}><code>hsl</code></div>

</Flex>

<Flex>
